써도 되는 재화 (적극 사용)
  • 캐피탈: 기체 개발·강화에 대량 소모되지만, 스테이지 점령·퀘스트 보상으로 꾸준히 수급됨. 아끼기보다는 적극 투자
  • GP: 옵션파츠 구매 등에 사용. 캐피탈 1,000 사용 시 GP 20 획득 가능. 해체로도 수급
  • 일반 강화 소재: 강화 육성 스테이지에서 꾸준히 파밍 가능. 주력기 강화에 아끼지 말 것
  • 저랭크 기체 경험치: 개발 시 레벨 계승되므로 R/N 기체에 경험치 투자해도 손해 아님
효율적인 자원 운용 체크리스트
  • 매주 캐릭터 리퀘스트 위클리 완료 → SP칩 20개 확보
  • 이벤트 공지 확인 → SSP칩·한정 SP칩 무료 배포 놓치지 않기
  • 마스터리그 시즌 보상 챙기기 → SP칩, ULT 재료 확보
  • 강화 육성 스테이지 횟수는 클리어 가능한 최고 레벨에 투자
  • 불필요 기체는 "알아서 해체"로 정리 → GP 수급
  • 스테이지 클리어 시 점령지 최대한 확보 → 캐피탈 추가 수급
  • 개발 계통도 미리 확인 → 불필요한 기체에 희소 재화 투자 방지
무과금 플레이어 핵심 전략: SP칩은 약 5주에 1기 SP화 가능한 페이스. 조급해하지 말고 확실한 기체에만 투자. 시리즈/태그 한정 SP칩 이벤트를 활용하면 범용 칩을 아낄 수 있다. SSP칩은 당분간 최최우선 기체에만 사용하고 나머지는 저축.

  • 대형 기체도 2x2 기구 아이콘이 있는 경우만 2칸 차지, 나머지는 1칸
  • 콘텐츠마다 부대를 자주 바꿔야 하므로 부대 저장 기능 적극 활용 (최대 10개 저장)
  • 기체 보유 상한은 1주년 업데이트로 1,200기 → 1,400기로 확장
부대 종류: 시리즈 부대 vs 프리 부대
  • 시리즈 부대: 현재 플레이 중인 건담 시리즈에 등장하는 유닛과 캐릭터만 편성 가능. 스토리 스테이지 공략 시 사용
  • 프리 부대: 어떤 시리즈의 유닛과 캐릭터든 자유롭게 편성 가능. 이터널 로드, 맵 이벤트, 마스터 리그 등에서 사용
시리즈 부대의 함정 — 스토리 진행 시 해당 시리즈 유닛만 쓸 수 있으므로, 특정 시리즈에만 올인하면 다른 시리즈 스토리에서 막힌다. 각 시리즈별로 최소 공격형 1기 + 내구형 1기는 확보해 둘 것.
파일럿-기체 시너지
  • 스킬·어빌리티에 의한 능력치 증가가 퍼센트(%)로 적용되므로, 기본 스탯이 높은 UR/SSR 조합에서 차이가 극대화됨
  • 같은 시리즈 파일럿 + 기체 조합 시 시리즈 보너스 발동 → 추가 능력치 상승
  • 파일럿 어빌리티 효율은 등급이 올라갈수록 대폭 상승 → UR/SSR 파일럿 확보가 핵심
  • 기체 타입(공격/방어/지원)과 파일럿 스탯 방향이 일치해야 효율 극대화
예시: UR 키라 야마토 + 스트라이크 프리덤(EX) = SEED 시리즈 보너스 + 공격형 시너지 최적. UR 아무로 레이 + 뉴건담(EX) = 퍼스트건담/역습의 샤아 이중 태그 활용 가능.

서포터 시스템
서포터 = 편성 핵심

서포터는 부대 전체 능력치를 올려주고, 특정 태그 기체에 추가 버프를 제공한다. SSR 이상 서포터 효과가 압도적이므로 보유 서포터 기준으로 부대를 구성하는 것이 정석이다.

  • 각 소대에 서포터 1기씩 배치 가능
  • 서포터의 리더 스킬이 특정 태그를 대상으로 능력치 상승 버프 제공
  • 리더 스킬 대상 태그에 해당하는 유닛을 최대한 많이 편성해야 효율 극대화
  • 서포터 범위 스킬(회복, 버프 등)은 전투 중 독립적으로 사용 가능

  • 서포터의 리더 스킬은 1차 태그(작품 태그)와 2차 태그(특성 태그) 조합으로 효과 범위가 결정됨
  • 예: 라크스 서포터 → 'SEED' + 'SEED DESTINY' 양쪽 시리즈 유닛에게 버프 적용
  • 예: 레인 서포터 → 'G 건담' 시리즈 + '격투' 특성 태그 유닛에게 강력한 공격 버프
  • 리더 스킬 대상 태그가 겹치는 유닛을 5기 모두 맞추면 풀 시너지 달성
주의 — 리더 스킬 대상 태그에 해당하지 않는 유닛은 서포터 버프를 받지 못한다. 강력한 유닛이라도 태그가 맞지 않으면 편성 효율이 크게 떨어지므로, 항상 서포터 대상 태그부터 확인할 것.

스토리 스테이지
  • 해당 시리즈 유닛만 사용 가능 (시리즈 부대)
  • 각 시리즈별 공격형 에이스 1기 + 내구형 3~4기로 구성하면 안정적
  • 에이스 기체로 반격 딜링, 나머지는 방패 역할
  • SR 이상 기체 1기만 집중 육성해도 대부분의 스토리 클리어 가능
이터널 로드
  • 태그 제한이 핵심 — 출격 가능 태그를 사전에 확인하고 유닛 육성 계획 수립
  • 맵병기 기체 적극 활용하여 잡몹 일소
  • 후반부로 갈수록 난이도 급상승 → 내구형 방패 + 에이스 딜러 조합 필수

  • EX 난이도는 풀 UR 편성 + 서포터 시너지 필수
마스터 리그 (PvP)
  • 프리 부대 사용 → 최강 유닛으로 편성
  • 시즌마다 특정 태그에 보너스 부여 → 해당 태그 유닛으로 편성하면 유리

  • 상대 편성 확인 후 약점 속성 유닛으로 카운터 편성
  • 시즌 10 (2026년 3월~) 진행 중
맵 이벤트
  • 기간 한정 이벤트로 보상이 풍부 → 최우선 공략 대상

  • 보스 스테이지에서는 지원방어 라인 끊기 전략 활용

  • UR 라크스 보상 획득을 위해 반드시 공략해야 하는 콘텐츠

고기방패 전략이란?

모든 시리즈 유닛을 골고루 육성하기 어려운 초~중반 유저를 위한 효율적 편성법이다.

  • 핵심 원리: 강력한 SSR/UR 공격기체 1~2기를 집중 육성하여 반격으로 적을 소탕
  • 나머지 3~4 슬롯은 내구(방어) 타입 기체로 채워서 1회용 방패로 활용
  • 기체는 개발(생산)로 양산 가능 → 기체 확보보다 시리즈별 내구 타입 캐릭터 확보가 관건
고기방패 편성 구체적 방법
  1. 에이스 유닛 선정: 보유 중인 가장 강력한 UR/SSR 공격형 기체 + 파일럿 조합을 최대한 육성한다
  2. 방패 유닛 제작: 해당 시리즈의 SR/R급 내구형 기체를 개발로 양산한다. HP와 장갑이 높은 기체 우선
  3. 방패 캐릭터 배치: 어빌리티가 방어·HP 관련인 캐릭터를 방패 유닛에 태운다. 어빌리티 레벨은 낮아도 무방
  4. 진형 배치: 에이스를 후방에 배치하고 방패 유닛을 전방에 배치. 적의 공격을 방패가 흡수하면 에이스가 반격으로 처리
  5. 지원방어 활용: 방패 유닛이 에이스 대신 지원방어를 받아주면 에이스의 생존율이 크게 상승
주의사항 — 방패 유닛이 전멸하면 에이스에게 공격이 집중되므로, 에이스 유닛의 회피 또는 방어 어빌리티도 어느 정도 확보해 둘 것. 또한 방패 유닛도 최소한의 공격력이 있어야 반격으로 소량이나마 딜을 넣을 수 있다.

지원방어 상세
  • 인접한 아군이 공격당할 때 지원방어를 선언하면, 대상이 받을 피해를 대신 받는다

지원공격 상세

  • 경감 없이 원래 공격력 그대로 적용 → 실질 화력 대폭 상승
  • 아군 턴·적 턴(반격 시) 모두 사용 가능
  • 보스전에서 지원공격을 몰아넣으면 한 턴에 집중 딜링 가능
  • 지원공격 횟수도 턴당 제한 있음 — 가장 높은 딜이 나오는 타이밍에 사용
보스전 핵심 전략 — 후반 보스들은 지원방어를 3회까지 사용하는 친위대를 대동한다. 적 사이에 끼어들어 지원방어 라인을 끊은 후 지원공격을 집중하는 것이 공략 포인트다. 보스의 친위대를 먼저 제거하거나, 진형 사이에 아군을 밀어 넣어 라인을 차단하라.

텐션(Tension) 시스템
텐션 기본 메커니즘

텐션은 파일럿의 전투 의지를 수치화한 것으로, MP를 소모하는 무기(맵병기 제외)의 데미지에 직접 영향을 준다. 텐션이 높을수록 데미지가 증가하고, 낮을수록 감소한다.

  • 텐션 상승: 적을 격파했을 때, 적에게 MP 소모 무기로 공격 성공 시 (소모 MP보다 적은 MP를 획득하지 않는 경우)
  • 텐션 하락: MP 소모가 획득보다 클 때, 맵병기(MP 소모) 사용 시, 공격 빗나감, 피격 시
  • 텐션 단계: 최저 → 낮음 → 보통 → 높음 → 최고 (5단계)
텐션 활용 전략
  • 텐션 올리기: 약한 적부터 처리하여 격파 보너스로 텐션을 올린 뒤 보스에게 집중
  • 적 텐션 깎기: 사거리 밖에서 원거리 공격으로 적 보스의 텐션을 낮춰 놓으면 받는 피해가 줄어듦
  • 맵병기 텐션 관리: 맵병기는 MP를 대량 소모하여 텐션이 하락하므로, 잡몹 소탕 후 텐션 회복 시간을 고려할 것
  • 회피형 유닛 활용: 적 공격을 회피하면 텐션이 떨어지지 않으므로, 회피 스탯이 높은 유닛은 텐션 유지에 유리
고급 팁 — 보스 직전 웨이브에서 의도적으로 약한 적을 남겨두고 텐션을 최고까지 올린 뒤, 보스전에 진입하면 첫 턴부터 최대 화력을 뽑을 수 있다.

지형 적성 시스템
지형 종류와 적성

스테이지마다 지상/우주/공중(수상) 등의 지형이 설정되며, 유닛별로 각 지형에 대한 적성 등급(S/A/B/C/D)이 있다. 적성이 낮은 지형에서는 능력치가 하락한다.

지형설명유리한 유닛
지상 가장 흔한 지형. 대부분의 유닛이 보통 이상 적성 육전형 건담, 이즈루하 진영 등 지상전 특화 기체
우주 역습의 샤아, UC, 0083 등 우주전 스테이지. 지상 전용 기체는 큰 페널티 뉴건담, 사자비, GP03 등 우주 적성 S~A 기체
공중/수상 일부 특수 스테이지. 비행 가능 또는 수륙양용 기체가 유리 Z건담(변형 가능), 수중형 MS 등
지형 적성 활용
  • 출격 전 스테이지 지형을 반드시 확인 — 지형 아이콘이 표시됨
  • 적성 S~A 유닛을 우선 편성하고, B 이하 유닛은 가급적 피할 것
  • 같은 기체라도 개량/변형으로 지형 적성이 변하는 경우가 있음
  • 일부 파일럿 어빌리티가 특정 지형에서 추가 능력치 보너스 제공
  • 지형에 의한 이동 제한도 존재 — 지상 전용 기체는 우주 맵에서 이동력 감소
실수 방지 — 우주 스테이지에 지상 적성만 높은 기체를 출격시키면 공격력·방어력·명중률·회피율 모두 크게 하락한다. 특히 이터널 로드 고난도에서 이 실수를 하면 클리어 불가능해질 수 있다.

턴 순서와 행동 시스템
턴 진행 구조
  • 전투는 아군 페이즈 → 적군 페이즈 교대 턴제로 진행
  • 아군 페이즈에서 모든 아군 유닛이 순서대로 이동 + 행동
  • 유닛의 기동력(스피드) 스탯이 행동 순서에 영향을 줌
  • 각 유닛은 턴당 이동 1회 + 행동 1회 수행 가능
  • 행동: 공격 / 대기 / 스킬 사용 / 변형 등
행동 순서 전략
  • 지원형 먼저 행동: 버프 스킬을 먼저 사용한 뒤 공격형이 공격하면 버프 적용 상태로 딜링 가능
  • 맵병기 먼저 사용: 잡몹을 맵병기로 소탕한 뒤 남은 유닛이 보스에게 집중
  • 탱커 위치 선점: 방어형 유닛을 먼저 이동시켜 에이스 앞에 배치하면, 적 턴에서 지원방어 가능
  • 반격 타이밍 고려: 적 턴에 반격이 일어나므로, 반격 화력이 높은 유닛은 적과 인접하게 배치

  • 유인 전술: 내구형 유닛을 전방에 내보내 적을 유인한 뒤, 후방 사격 유닛으로 집중 사격

10층 — 가더 기믹 본격화
기믹: 맵에 공격/방어 버프 가더 등장. 점거 시 해당 스탯 대폭 상승.

11층 — 집중 화력전
  • 가더 기믹은 10층과 유사하나 적 HP/공격력 상승
  • 적 엘리트 2기가 맵병기 보유 → 맵병기 적을 1턴 내 우선 처치
  • 맵병기 적 처치 후 보스에 집중
  • 2부대로 양쪽 엘리트 동시 처리하는 분할 전략 유효
12층 — 복합 가더 + 턴 제한
  • 턴 제한이 추가됨 — 느긋하게 진행 불가
  • 공격 버프 가더를 빠르게 점거하고 즉시 보스 공격 개시
  • 보스의 지원 방어 유닛이 등장 → 맵병기로 보스 + 친위대 동시 타격 추천
  • DPS 부족 시 서포터의 공격 버프를 중첩하여 화력 보강
실패 원인: 턴 제한 내 보스를 처치하지 못하는 경우 대부분 가더 미점거 또는 화력 분산이 원인.

    • 단, 샤이닝 건담 방어력이 약함 → 반드시 탱커로 보호

    • 다음 턴: 마스터건담 처리
    • 2부대 DPS가 부족하면 데빌건담을 1턴에 못 잡아 실패 확률 급상승

  • 최종 보스 격파 후에는 스코어 어택 모드가 개방 -- 이터널 로드 익스트림급에 맞먹는 난이도
  • 이벤트 기간 한정이므로 효율적인 진행이 중요

핵심 메커니즘 상세
태그 시스템과 출격 제한
  • 작품 태그: 기동전사 Z건담, 신기동전기 건담W, 기동전사 건담SEED 등 시리즈별 태그
  • 특성 태그: 돌파력, 견고, 기동, 화력, 지원 등 유닛의 역할 태그
  • 각 스테이지에서 요구하는 태그 조건을 만족하지 못하면 출격 자체가 불가
  • 일부 스테이지는 작품 태그 + 특성 태그를 동시에 요구 -- 조건이 까다로움
  • 이벤트 부스트 기체(해당 이벤트 픽업 유닛)는 스탯 보너스 + 이벤트 포인트 추가 획득
버프 시스템
  • 맵 위의 특정 칸에서 버프 아이템을 획득할 수 있음
  • 버프 종류: 공격력 UP, 방어력 UP, SP 게이지 가속 등
  • 버프를 최대한 많이 모은 상태에서 보스에 도전하면 유리
  • 이중 버프: 작품 태그와 특성 태그 양쪽에 해당하는 유닛은 버프 효과가 중첩
  • 루트 선택 시 버프 칸을 우선 경유하는 것이 효율적
맵 구조와 루트 선택
  • 맵은 여러 지역(Area)으로 나뉘며, 지역마다 요구 태그가 다름
  • 일부 칸은 갈림길이 있어 루트 선택이 가능
  • 쉬운 루트: 보스까지 빠르게 도달, 버프 적음
  • 우회 루트: 시간은 더 걸리지만 버프와 보상을 더 많이 획득 가능
  • 초보자는 먼저 쉬운 루트로 클리어 후, 여유 있을 때 우회 루트 도전 추천
주의: 지역3 TR-5 스테이지

사전 육성 필수 태그 (우선순위)
  1. 기동전사 Z건담 -- 맵 이벤트에서 가장 자주 요구되는 작품 태그. Z건담, 백식, 큐베레이 등 확보 권장
  2. 신기동전기 건담W -- W 시리즈 기체를 3~4기 이상 SP화해두면 안정적
  3. 돌파력 + 견고 -- 가장 범용적인 특성 태그 조합. 대부분의 맵에서 활용 가능
  4. 기동전사 건담SEED -- SEED 시리즈도 빈번하게 등장. 프리덤, 저스티스 계열 육성

  • 부대 저장 기능 필수 활용: 태그마다 다른 편성이 필요하므로, 주요 태그별로 부대를 미리 저장해두면 교체가 빠름
  • 최소 3~4개 시리즈의 부대를 사전 저장 추천
  • 이벤트 부스트 기체는 반드시 편성에 포함 -- 포인트 효율이 크게 차이남
  • 서포터도 태그에 영향받으므로, 서포터 태그도 확인할 것
  • 같은 시리즈 유닛끼리 편성하면 시리즈 보너스 발동 가능
